# Security Policy

## Supported Versions

| Version | Supported |
|---------|-----------|
| 0.1.x   | Yes       |

## Reporting a Vulnerability

If you discover a security vulnerability, **do not open a public issue**. Instead, email arturoornelas62@gmail.com with:

- Description of the vulnerability
- Steps to reproduce
- Potential impact

You will receive a response within 72 hours. We will work with you to understand and address the issue before any public disclosure.
